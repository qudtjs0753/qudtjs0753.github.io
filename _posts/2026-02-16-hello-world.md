## 블로그를 시작합니다

안녕하세요! 이 블로그의 첫 번째 포스트입니다.

Jekyll과 Chirpy 테마를 사용하여 GitHub Pages에 블로그를 만들었습니다.

### 이 블로그에서는

- 개발하면서 배운 것들
- 문제 해결 과정
- 다양한 기술 관련 이야기

를 기록할 예정입니다.

### 사용한 기술 스택

| 도구 | 역할 |
|------|------|
| Jekyll | 정적 사이트 생성기 |
| Chirpy | 블로그 테마 |
| GitHub Pages | 호스팅 |
| Markdown | 글 작성 형식 |

앞으로 꾸준히 기록해 나가겠습니다.
